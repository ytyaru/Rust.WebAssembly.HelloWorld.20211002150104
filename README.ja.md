## rust

```sh
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

# 使い方

```sh
git clone https://github.com/ytyaru/Rust.WebAssembly.HelloWorld.20211002150104
cd Rust.WebAssembly.HelloWorld.20211002150104
cargo build
cargo run
```
